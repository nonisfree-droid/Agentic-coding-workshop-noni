# Kitchen Passport

냉장고 속 재료로 세계 요리를 만들어보는 레시피 모험 앱.

## Overview

Fridge Recipe Adventure는 사용자가 냉장고에 있는 재료(고기, 향신료 등)를 선택하면, 그 재료로 만들 수 있는 세계 각국의 요리를 추천해주는 서비스다.

## User Flow

### 1단계: 재료 선택

사용자는 두 가지 카테고리에서 재료를 선택한다:

#### 1-1) 고기 종류
- 닭
- 돼지
- 소고기
- 해산물

#### 1-2) 향신료
- 바질
- 오레가노
- 타임
- 파프리카

선택 완료 후 다음 단계로 진행.

### 2단계: 요일별 메뉴 추천

선택한 재료를 기반으로 이번주 요일별 메뉴를 추천한다:

| 요일 | 날짜 | 추천 메뉴 | 국가 |
|------|------|-----------|------|
| 수 | 7/1 | 지중해식 샐러드 | 지중해 |
| 목 | 7/2 | 이탈리아 파스타 | 이탈리아 |
| 금 | 7/3 | 그리스 그릭 | 그리스 |
| 토 | 7/4 | 스페인 따뚜르 | 스페인 |
| 일 | 7/5 | 터키 케밥 | 터키 |

### 3단계: 레시피 상세 + 구매 링크

선택한 메뉴의 상세 레시피를 보여준다:

- **레시피 이름**
- **재료 목록** (보유 여부 표시)
  - 보유 ✅
  - 부족 ❌ → 구매 링크 제공
- **인스타/이미지** 참고 사진
- **구매하기 버튼** (부족한 재료 쿠팡/마켓컬리 링크)

## Technical Stack (예정)

- Frontend: React / Next.js (또는 Flutter)
- Backend: Python (FastAPI) / Node.js
- DB: PostgreSQL / SQLite
- AI: 레시피 추천 및 재료 매칭
